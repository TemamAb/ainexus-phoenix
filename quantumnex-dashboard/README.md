# QuantumNex Dashboard (skeleton)\n\nThis is a scaffold for the QuantumNex enterprise DeFi dashboard. It contains TypeScript + Next.js skeleton components, hooks and services to be extended.\n\nQuickstart:\n1. npm install\n2. copy .env.example to .env.local and set RPC keys\n3. npm run dev\n\nFeatures included (stubs): wallet integration, websocket streaming, optimization engine scheduler, components and theme.\n\nSecurity: No secrets are included. Use environment variables for RPC keys and provider secrets.
